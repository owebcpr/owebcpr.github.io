Трансформація елементів. Властивість transform

1. Що таке трансформація?

Трансформація дозволяє змінювати зовнішній вигляд елементів: обертати, переміщувати, масштабувати та нахиляти їх.

2. Основні види трансформації

transform: translate()

Переміщує елемент по горизонталі та вертикалі:


div {
  transform: translate(50px, 20px); /* X, Y */
  transform: translateX(100px);     /* тільки по X */
  transform: translateY(-30px);     /* тільки по Y */
}

transform: rotate()

Обертає елемент на заданий кут:


div {
  transform: rotate(45deg);     /* градуси */
  transform: rotate(0.5turn);   /* оберти */
  transform: rotate(3.14rad);   /* радіани */
}

transform: scale()

Масштабує елемент:


div {
  transform: scale(1.5);        /* X і Y */
  transform: scale(2, 0.5);     /* X, Y */
  transform: scaleX(1.2);       /* тільки X */
  transform: scaleY(0.8);       /* тільки Y */
}

transform: skew()

Нахиляє елемент:


div {
  transform: skew(30deg);       /* X і Y */
  transform: skew(20deg, 10deg);/* X, Y */
  transform: skewX(15deg);      /* тільки X */
  transform: skewY(-10deg);     /* тільки Y */
}

Комбіновані трансформації

Можна комбінувати кілька трансформацій:


div {
  transform: translate(50px, 20px) rotate(45deg) scale(1.2);
}

4. Додаткові властивості

transform-origin

Визначає точку, відносно якої відбувається трансформація:


div {
  transform-origin: center;     /* за замовчуванням */
  transform-origin: top left;   /* лівий верхній кут */
  transform-origin: 50px 30px;  /* конкретні координати */
}

transition

Додає плавність анімації трансформації:


div {
  transition: transform 0.3s ease-in-out;
}
div:hover {
  transform: rotate(45deg);
}
