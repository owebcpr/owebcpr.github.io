Форми в HTML

Форми в HTML використовуються для збору інформації від користувачів. Наприклад, реєстрація, входи, голосування та інші дії — все це відбувається через форми.

Текстові поля

Для введення тексту використовують тег <input> з атрибутом type="text".

<input type="text" placeholder="Введіть ім'я" name="name">

Поле для введення електронної пошти

<input type="email" placeholder="email@example.com" name="email">

Многострокове поле

Для введення довгого тексту, наприклад коментаря, використовують тег <textarea>.

<textarea name="comment" rows="4" cols="40" placeholder="Ваш коментар..."></textarea>

Радіокнопки

Радіокнопки дають змогу обрати лише один варіант.


<input type="radio" name="gender" value="male" checked> Чоловік
<input type="radio" name="gender" value="female"> Жінка

Прапорці (checkbox)

Можна обрати кілька варіантів одразу.


<input type="checkbox" name="hobby" value="reading"> Читання
<input type="checkbox" name="hobby" value="sports" checked> Спорт

Групування елементів

Для зручності елементи можна групувати за допомогою тегів <fieldset> і <legend>. 


<fieldset>
  <legend>Контактна інформація</legend>
  <input type="text" name="name" placeholder="Ім'я">
  <input type="email" name="email" placeholder="Email">
</fieldset>
Контактна інформація

Списки вибору

Для створення випадаючого списку використовують тег <select>. Атрибут multiple дозволяє вибрати кілька варіантів. 


<select name="language" multiple>
  <option value="html" selected>HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
</select>

Кнопки

Форми надсилаються за допомогою кнопок. Тег <button> може містити текст, іконки або HTML-елементи.


<button type="submit">Надіслати</button>
<button type="reset">Очистити</button>

Атрибути форм

  • placeholder — підказка в полі введення
  • value — значення за замовчуванням
  • checked — позначка для радіокнопок і прапорців
  • name — назва елемента (потрібна для обробки на сервері)
