Прості та вкладені селектори. Пріоритет селекторів

Селектор — це спосіб вказати, до яких елементів HTML слід застосувати певні стилі.

Прості селектори

До простих селекторів належать:

  • Селектори тегів — застосовуються до всіх елементів певного типу.
  • Селектори класів — застосовуються до елементів з певним класом (через .).
  • Селектори ідентифікаторів — застосовуються до елемента з певним id (через #). 

/* Селектор тега */
p {
  color: green;
}

/* Селектор класу */
.highlight {
  background-color: yellow;
}

/* Селектор ідентифікатора */
#main-title {
  font-size: 28px;
}

Вкладені селектори

Це селектори, які працюють з урахуванням вкладеності елементів.


/* Всі span всередині елемента з класом .info */
.info span {
  color: blue;
}

Пріоритет селекторів

Якщо до одного елемента застосовуються стилі з кількох джерел — вирішальним є пріоритет:

  1. Інлайн-стилі (найвищий пріоритет)
  2. Ідентифікатор (#id)
  3. Клас, атрибут, псевдоклас
  4. Тег (найнижчий пріоритет)

Приклад з конфліктом:



<h2 class="big" id="title" >Привіт!</h2>


            

/* обидва правила застосовуються до h2, але id має вищий пріоритет */
.big {
  color: green;
}

#title {
  color: red;
}

У цьому випадку заголовок буде червоним, бо селектор #title має вищий пріоритет, ніж .big.
