Способи завдання стилів. Каскадні таблиці стилів

Що таке стилі й навіщо вони потрібні?

CSS (Cascading Style Sheets) — це мова стилів, що дозволяє оформлювати HTML-документи: змінювати кольори, шрифти, відступи, розміщення елементів тощо. Стилі відокремлюють зміст від вигляду й допомагають створити привабливі, адаптивні сайти.

Як можна додавати стилі?

  • Inline-стилі: додаються безпосередньо в HTML через атрибут style.
  • Внутрішні стилі: пишуться в блоці <style> у розділі <head>.
  • Зовнішній файл: підключається за допомогою <link>.

Приклади:

<!-- Inline-стиль -->
<h1 style="color: red;">Привіт!</h1>

<!-- Внутрішній стиль -->
<style>
  p {
    color: green;
    font-size: 18px;
  }
</style>

<!-- Зовнішній стиль -->
<link rel="stylesheet" href="style.css">

Синтаксис CSS

селектор {
  властивість: значення;
}

/* приклад */
h2 {
  color: #00557f;
  font-weight: bold;
}

Чому таблиця стилів — каскадна?

CSS називається "каскадною" таблицею стилів, бо вона враховує порядок, специфічність та пріоритет правил. Це дозволяє вирішувати конфлікти між різними джерелами стилів:

  1. Inline-стилі мають найвищий пріоритет
  2. Далі — внутрішні стилі (в <style>)
  3. Найнижчий пріоритет — у зовнішніх файлів
