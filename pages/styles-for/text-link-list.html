CSS: Стилі для тексту, посилань, списків

1. Стилі для тексту

Можна змінювати колір, розмір, жирність, стиль тексту, вирівнювання та інше.


p {
  color: red;
  font-size: 18px;
  font-style: italic;
  font-weight: bold;
  text-align: center;
}

2. Стилі для посилань

Можна оформити зовнішній вигляд посилань у різних станах:


a {
  color: lightblue;
  text-decoration: none;
}
a:hover {
  color: yellow;
}
a:visited {
  color: gray;
}

3. Стилі для списків

Списки можна стилізувати, змінювати маркери, відступи тощо:


ul {
  list-style-type: square;
  padding-left: 20px;
  color: orange;
}
ol {
  list-style-type: upper-roman;
  color: lightgreen;
}
