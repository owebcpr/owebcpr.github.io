Код безпеки: веб-розробка vs кіберзлочинність

Виховний захід для гуртка "Web-програмування" з профілактики правопорушень

Про захід

Цей захід розроблений спеціально для юних веб-розробників, щоб навчити розрізняти правомірні та протиправні дії в інтернеті та показати зв'язок між веб-розробкою та кібербезпекою.

Інтерактивна розминка

Питання: Як ви розумієте слово "правопорушення" в інтернеті?

Обговорення: Чи може веб-розробник стати випадковим злочинцем?

Чорний та білий код

Розглянемо приклади корисного та шкідливого коду:

Добрий код: Створення освітнього сайту для школярів
// Код для створення навчального веб-додатку
function createEducationalApp() {
  // Допомагаємо учням вивчати програмування
  return "Безпечний навчальний ресурс";
}
Шкідливий код: Використання скриптів для крадіжки даних
// Код для викрадення логінів/паролів (НЕПРАВОМІРНО!)
function stealCredentials() {
  // Це кіберзлочинність!
  return "Ваші дані вкрадені";
}

Практичне завдання

Перед вами два фрагменти коду. Визначте, який з них правопорушний та чому?

Цифровий слід

Кожна ваша дія в інтернеті залишає слід. Розглянемо приклад:

Історія підлітка:
  1. Виклав у відкритий доступ чужі фото без дозволу
  2. Скачав піратський ігровий мод
  3. Написав скрипт для автоматизації лайків

Обговорення: Які наслідки можуть бути? (Штрафи, заблоковані акаунти, кримінальна відповідальність)

Як захистити себе та інших?

Правила "кодексу честі" веб-розробника:

  • Не тестувати "круті скрипти" на чужих сайтах
  • Перевіряти джерела бібліотек (чи не містять вірусів)
  • Повідомляти дорослим про підозрілі прохання в мережі
  • Використовувати свої навички для створення корисних продуктів
