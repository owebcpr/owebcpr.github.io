Веб-шрифти: Як зробити текст красивим

1. Що таке веб-шрифти?

Веб-шрифти - це спеціальні шрифти, які завантажуються з інтернету і відображаються на веб-сторінках. На відміну від звичайних шрифтів, які повинні бути встановлені на комп'ютері користувача, веб-шрифти працюють навіть якщо у людини немає цього шрифту!

2. Де знайти веб-шрифти?

Існує багато безкоштовних сервісів з веб-шрифтами:

  • Google Fonts - найпопулярніша безкоштовна колекція
  • Fonts.com - платні та безкоштовні шрифти
  • Adobe Fonts - для користувачів Adobe Creative Cloud
  • Font Squirrel - безкоштовні шрифти високої якості
  • DaFont - велика колекція безкоштовних шрифтів
  • 1001 Fonts - ще один популярний ресурс

3. Як підключити веб-шрифти?

Найпростіший спосіб - через Google Fonts:

Спосіб 1: Через посилання в HTML


<!-- Додаємо в розділ head -->
<link href="https://fonts.googleapis.com/css2?family=Roboto+Slab&display=swap" rel="stylesheet">

Спосіб 2: Через import в CSS


@import url('https://fonts.googleapis.com/css2?family=Comic+Neue&display=swap');

4. Як застосовувати веб-шрифти на сторінці?

Після підключення шрифту, використовуйте його в CSS:


body {
  font-family: 'Roboto Slab', serif;
}

h1, h2, h3 {
  font-family: 'Comic Neue', cursive;
  font-weight: bold;
}
