Блокові та рядкові елементи

HTML-елементи поділяються на два основних типи: блокові та рядкові.

Блокові елементи

Блокові елементи займають всю ширину свого батьківського елемента і завжди починаються з нового рядка. Приклади блокових елементів:

  • <div> — універсальний блок;
  • <p> — абзац тексту;
  • <h1>-<h6> — заголовки різних рівнів;
  • <ul>, <ol> — списки.

<div>
    <h2>Заголовок</h2>
    <p>Це абзац всередині блоку.</p>
</div>

Рядкові елементи

Рядкові елементи займають тільки необхідну ширину і не починаються з нового рядка. Приклади:

  • <span> — універсальний рядковий контейнер;
  • <a> — гіперпосилання;
  • <strong>, <em> — виділення тексту.

<p>Це приклад з <strong>жирним</strong> словом.</p>

Семантичні теги

Семантичні теги описують зміст елемента, що полегшує розуміння структури сторінки як для людей, так і для пошукових систем.

  • <header> — заголовок сторінки або розділу;
  • <nav> — навігаційне меню;
  • <main> — основний вміст сторінки;
  • <section> — розділ вмісту;
  • <article> — самостійний матеріал або публікація;
  • <footer> — нижній колонтитул сторінки або розділу.

<article>
    <header>
        <h2>Новина дня</h2>
    </header>
    <p>Сьогодні ми вивчаємо HTML.</p>
</article>

Emmet-скорочення

У редакторі Visual Studio Code можна використовувати скорочення Emmet. Наприклад:


div.container>ul>li*3

Це скорочення створить наступну структуру:


<div class="container">
    <ul>
        <li></li>
        <li></li>
        <li></li>
    </ul>
</div>
