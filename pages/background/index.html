Фонові зображення. Властивості background

1. Що таке фонові зображення?

Фонові зображення дозволяють додавати картинки на задній план елементів. Це можуть бути фотографії, текстури, графічні елементи або шаблони.

2. Основні властивості background

background-image

Вказує шлях до зображення:


div {
  background-image: url('шлях/до/зображення.jpg');
}

background-size

Визначає розмір фонового зображення:


div {
  background-size: cover;     /* покриває всю область */
  background-size: contain;   /* вміщає все зображення */
  background-size: 100% 100%; /* розтягує на всю область */
  background-size: 200px 150px; /* конкретні розміри */
}

background-position

Визначає позицію зображення:


div {
  background-position: center;    /* по центру */
  background-position: top left;  /* в лівому верхньому куті */
  background-position: 50% 50%;   /* координати */
  background-position: 20px 10px; /* відступи */
}

background-repeat

Визначає, чи повторюватиметься зображення:


div {
  background-repeat: no-repeat; /* не повторюється */
  background-repeat: repeat;    /* повторюється */
  background-repeat: repeat-x;  /* повторюється по горизонталі */
  background-repeat: repeat-y;  /* повторюється по вертикалі */
}
