Теги – як основа HTML

HTML-сторінки будуються з тегів. Теги — це інструкції для браузера, які пояснюють, як саме відображати вміст.

Основні теги виглядають ось так:


<p>Це абзац</p>
<h1>Заголовок</h1>
<a href="https://example.com">Посилання</a>

Основні теги HTML

  • <html>...</html> — кореневий елемент HTML-документа.
  • <head>...</head> — містить службову інформацію про документ (мета-дані, заголовок, стилі).
  • <title>...</title> — назва сторінки, яка відображається у вкладці браузера.
  • <body>...</body> — основний вміст сторінки.
  • <h1>...</h1> до <h6>...</h6> — заголовки різного рівня.
  • <p>...</p> — абзац тексту.
  • <a href="...">...</a> — гіперпосилання.
  • <img src="..." alt="..."> — зображення.
  • <ul>, <ol>, <li> — списки (неупорядковані та упорядковані).
  • <div>...</div> — універсальний блок.
  • <span>...</span> — вбудований елемент для стилізації частин тексту.

Теги можуть бути:

  • Парними — мають відкриваючий і закриваючий тег, як <p>...</p>
  • Одинарними — закриваються самі, як <br> або <img>

Атрибути тегів

Теги можуть мати атрибути — додаткову інформацію, яка уточнює дію тега.


<img src="cat.jpg" alt="Котик" width="300">

У цьому прикладі:

  • src — шлях до зображення
  • alt — текст, що з'являється, якщо зображення не завантажилось
  • width — ширина зображення

Emmet-скорочення

Emmet — це інструмент для швидкого набору коду в редакторі, наприклад, VSCode. Спробуй ввести таке скорочення:


ul>li*3

І натисни Tab — отримаєш:


<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
