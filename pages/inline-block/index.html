Блочні та рядкові елементи. Властивості блоків

1. Що таке блочні елементи?

Блочні елементи займають всю доступну ширину і починаються з нового рядка. Вони можуть містити інші блочні та рядкові елементи.


<div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <section>, <article>

2. Що таке рядкові елементи?

Рядкові елементи займають тільки необхідну ширину і не починають новий рядок. Вони зазвичай містять текст або інші рядкові елементи.


<span>, <a>, <strong>, <em>, <img>, <br>

3. Властивості блоків: Margin, Padding, Border

Що таке Box Model?

Кожен блочний елемент має так звану "модель коробки", яка складається з:

Margin
Border
Padding
Контент
Margin - зовнішній відступ (простір між елементами)
Border - рамка навколо елемента
Padding - внутрішній відступ (простір всередині, біля контенту)
Content - власне вміст елемента

Приклади використання:

Padding (внутрішній відступ)


div {
  padding: 20px; /* всі сторони */
  padding: 10px 20px; /* верх/низ | ліво/право */
  padding: 5px 10px 15px 20px; /* верх | право | низ | ліво */
}
Елемент з padding

Margin (зовнішній відступ)


div {
  margin: 30px; /* всі сторони */
  margin: 15px 30px; /* верх/низ | ліво/право */
  margin: 10px 20px 30px 40px; /* верх | право | низ | ліво */
  margin: 0 auto; /* центрування по горизонталі */
}

Border (рамка)


div {
  border: 2px solid black; /* товщина | стиль | колір */
  border-radius: 10px; /* закруглення кутів */
  border-top: 1px dotted red; /* окрема сторона */
}
