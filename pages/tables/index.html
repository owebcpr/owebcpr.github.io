Таблиці в HTML

Таблиця в HTML використовується для організації даних у рядки та стовпці. Вона створюється за допомогою тегу <table>.

Основні теги таблиць

  • <table> — контейнер для таблиці
  • <tr> — table row — рядок таблиці
  • <th> — table header — заголовок стовпця
  • <td> — table data — звичайна клітинка таблиці

Проста таблиця

<table>
  <tr>
    <th>Ім'я</th>
    <th>Вік</th>
  </tr>
  <tr>
    <td>Оля</td>
    <td>12</td>
  </tr>
  <tr>
    <td>Артем</td>
    <td>14</td>
  </tr>
</table>
Ім'я Вік
Оля 12
Артем 14

Атрибути таблиць

Деякі популярні атрибути:

  • border — товщина рамки таблиці (старий спосіб, краще використовувати CSS)
  • colspan — об’єднання кількох стовпців
  • rowspan — об’єднання кількох рядків

Таблиця з об’єднанням комірок

<table>
  <tr>
    <th rowspan="2">Ім'я</th>
    <th colspan="2">Контакти</th>
  </tr>
  <tr>
    <th>Email</th>
    <th>Телефон</th>
  </tr>
  <tr>
    <td>Марія</td>
    <td>maria@gmail.com</td>
    <td>123-456</td>
  </tr>
</table>
Ім'я Контакти
Email Телефон
Марія maria@gmail.com 123-456

Поради

  • Завжди використовуйте <thead>, <tbody>, <tfoot> для структурування складних таблиць.
  • Використовуйте CSS для стилізації таблиць.

Цікаво знати

На початку розвитку вебу, ще до появи сучасних підходів до верстки, таблиці активно використовували не лише для відображення даних, а й для створення макетів сторінок. Це дозволяло розміщувати елементи на екрані в рядках і стовпцях, подібно до сітки. Наприклад, «шапка» сайту могла бути одним рядком таблиці, «меню» — другим, а основний вміст — третім. Хоча це було незручно й не зовсім правильно з точки зору семантики, такий підхід довгий час був стандартом.

Сьогодні для створення макетів використовують CSS-технології — Flexbox, Grid, або позиціонування. Таблиці ж призначені винятково для структурованих даних.

Приклад верстки сторінки за допомогою таблиці

Це спрощений приклад того, як раніше створювали макет сторінки:

<table border="1" width="100%" cellspacing="0" cellpadding="10">
  <tr>
    <td colspan="2"><strong>Шапка сайту</strong></td>
  </tr>
  <tr>
    <td width="20%">Меню</td>
    <td>Основний вміст</td>
  </tr>
  <tr>
    <td colspan="2">Підвал</td>
  </tr>
</table>

У цьому прикладі шапка, меню, вміст і підвал сайту — це комірки таблиці. Зараз такі підходи вважаються застарілими, але вони відіграли важливу роль в історії веб-розробки.
