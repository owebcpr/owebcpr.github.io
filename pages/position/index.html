Позиціонування елементів. Властивість position

1. Що таке позиціонування?

Властивість position визначає, як елемент розташовується на сторінці. Коли ми застосовуємо позиціонування (relative, absolute, fixed), ми можемо використовувати властивості:

  • top - відступ зверху
  • right - відступ справа
  • bottom - відступ знизу
  • left - відступ зліва

2. Одиниці виміру: пікселі та відсотки

Властивості позиціонування можна задавати в різних одиницях:

Пікселі (px)

Фіксований розмір. 1px = 1 точка на екрані.


.element {
  position: absolute;
  top: 50px;    /* 50 пікселів від верху */
  left: 100px;  /* 100 пікселів зліва */
}

Відсотки (%)

Відносний розмір. Відсотки рахуються від розміру батьківського контейнера:


.element {
  position: absolute;
  top: 10%;     /* 10% від висоти батька */
  left: 20%;    /* 20% від ширини батька */
}

Приклад: Якщо батьківський контейнер має ширину 500px, то left: 20% = 100px

2. position: relative

Елемент позиціонується відносно свого нормального положення. Залишається в потоці документа - інші елементи "бачать" його на старому місці.


.element {
  position: relative;
  top: 20px;
  left: 30px;
}

3. position: absolute

Елемент видаляється з нормального потоку і позиціонується відносно найближчого предка з позиціонуванням (не static).

Якщо батько має позиціонування:


.container {
  position: relative; /* Створює контекст */
}
.child {
  position: absolute; /* Позиціонується відносно .container */
  top: 0;
  left: 0;
}

Якщо батько НЕ має позиціонування:


.container {
  position: static; /* За замовчуванням - не створює контекст */
}
.child {
  position: absolute; /* Позиціонується відносно вікна браузера! */
  top: 0;
  left: 0;
}

Важливо: Інші елементи "не бачать" абсолютно позиціонований елемент і займають його колишнє місце!


.container {
  position: relative; /* Створюємо контекст */
}
.element {
  position: absolute;
  top: 50px;
  right: 40px;
}

4. position: fixed

Елемент позиціонується відносно вікна браузера (viewport). Залишається на одному місці при прокрутці.

Увага: Fixed завжди позиціонується відносно вікна браузера! Це лише імітація для навчання. 


.element {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
