Програмне забезпечення для роботи з web

Що таке сайти? Сайт — це набір веб-сторінок, які зберігаються на спеціальних комп’ютерах, які називаються веб-серверами. Щоб створити сайт, розробники пишуть код (найчастіше на мовах HTML, CSS, JavaScript), зберігають його у файли, і потім розміщують ці файли на сервері.

Коли ми в браузері вводимо адресу сайту (наприклад, google.com), браузер звертається до сервера, де зберігається сайт, і завантажує його сторінки, щоб ми могли їх переглядати.

Види сайтів:

  • Статичні сайти — прості сторінки з текстом, зображеннями, без складної взаємодії з користувачем.
  • Динамічні сайти — змінюють вміст залежно від дій користувача (наприклад, соціальні мережі, онлайн-магазини).

Для роботи з сайтами потрібне спеціальне програмне забезпечення:

  • Браузери — це програми, які дозволяють переглядати сайти. Приклади: Google Chrome, Mozilla Firefox, Microsoft Edge. Саме вони показують результат роботи HTML, CSS і JavaScript.
  • Браузери мають інструменти розробника. Щоб їх відкрити, натисніть F12 або правою кнопкою миші на сторінці → Перевірити елемент. Там можна побачити код сайту, структуру HTML, стилі CSS, подивитися помилки в JavaScript.
  • Локальні сервери — дозволяють запускати сайти «на своєму комп’ютері» під час розробки. Найпопулярніші: Live Server (розширення до VSCode), MAMP, LocalWP.
  • Редактори коду — програми для написання коду: VSCode, Sublime Text, Notepad++.
  • Системи контролю версій — наприклад, Git. Вони дозволяють зберігати історію змін у коді.

Ці інструменти допомагають створювати, перевіряти, запускати й публікувати сайти. На наступних заняттях ми докладніше розглянемо редактори коду та налаштування середовища для розробки.

Знайомство з редакторами коду

Код для вебсайтів можна писати навіть у звичайному Блокноті (Notepad), що є в кожному комп’ютері з Windows. Ось приклад простого HTML-файлу:

<!DOCTYPE html>
<html>
  <head>
    <title>Мій перший сайт</title>
  </head>
  <body>
    <h1>Привіт, світ!</h1>
  </body>
</html>

Цей код можна зберегти з розширенням .html (наприклад, index.html) і відкрити у браузері. Але працювати з більшими проєктами в Блокноті — незручно, оскільки немає підсвітки коду, автодоповнення чи навігації.

Тому розробники використовують редактори коду — спеціальні програми, які полегшують створення, читання й редагування коду. Вони мають:

  • Підсвітку синтаксису
  • Автоматичне завершення коду
  • Можливість встановлювати розширення та плагіни
  • Інтеграцію з терміналом та системою контролю версій

Існує чимало популярних редакторів коду:

  • Sublime Text — дуже легкий і швидкий, але багато функцій доступні лише у платній версії.
  • Atom — зручний редактор від GitHub, однак розвиток проєкту припинився у 2022 році.
  • Brackets — орієнтований на веброзробку, має live preview, але вже застарілий.
  • Notepad++ — розширений варіант Блокноту з підсвіткою та плагінами, але без повноцінної підтримки сучасних вебінструментів.

Серед усіх цих варіантів найкращим вибором для сучасної веброзробки є Visual Studio Code (VSCode).

Чому саме VSCode?

  • Безкоштовний та з відкритим вихідним кодом
  • Працює на Windows, macOS і Linux
  • Має вбудований термінал, Git, дебагер
  • Має величезну бібліотеку плагінів для HTML, CSS, JavaScript, SASS, React, PHP тощо
  • Зручний інтерфейс та підтримка тем оформлення

Visual Studio Code постійно оновлюється і використовується як професіоналами, так і початківцями.

Завантажити VSCode можна з офіційного сайту: code.visualstudio.com

Налаштування середовища програмування

Visual Studio Code (VSCode) — дуже потужний і зручний редактор. Він має багато функцій, які значно полегшують веброзробку. Давайте розглянемо, як з ним працювати.

Як створити файли та папки у VSCode:

  • Відкрий VSCode.
  • У меню обери File → Open Folder і вибери папку, в якій зберігатимеш проєкт.
  • У лівій панелі (Explorer) натисни правою кнопкою миші на порожньому місці → New File або New Folder.
  • Введи назву (наприклад, index.html, style.css) — і файл готовий до редагування.
  • Щоб відкрити файл — просто клікни по ньому у списку зліва.

Як встановлювати плагіни (розширення):

  • Натисни комбінацію клавіш Ctrl+Shift+X або натисни на іконку Extensions (у лівій панелі).
  • У полі пошуку введи назву розширення (наприклад: Live Server).
  • Натисни Install — плагін буде встановлено.

Корисні розширення для веброзробки:

  • All Autocomplete — розширює можливості автозаповнення, підказує навіть із сусідніх файлів.
  • Auto Close Tag — автоматично додає закриваючі HTML-теги.
  • Auto Complete Tag — підказує і закриває HTML- та JSX-теги.
  • Auto Rename Tag — змінює одночасно відкриваючий і закриваючий тег.
  • Code Runner — дозволяє запускати код прямо у VSCode (підтримує HTML, CSS, JS та інші).
  • JavaScript (ES6) code snippets — готові шаблони сучасного JavaScript-коду.
  • Live Server — відкриває сайт у браузері й автоматично оновлює його при зміні коду.
  • Multiple clipboards for VSCode — зберігає кілька буферів обміну (Ctrl+Shift+V).
  • Sass — підсвітка синтаксису SCSS/Sass.
  • Live Sass Compiler — автоматично компілює SCSS/SASS у CSS при збереженні файлу та створює CSS-файл у проекті.
  • Theme - Oceanic Next — темна тема з приємними кольорами (опціонально).
  • vscode-icons — додає гарні іконки до файлів і папок у провіднику.

Ці розширення значно підвищують зручність та швидкість роботи з кодом.

Робота з командами та скороченнями:

VSCode підтримує технологію Emmet — це скорочення, які розгортаються у повноцінний HTML-код. Наприклад:

ul>li*3 — створює список із трьох пунктів

Детальний список команд Emmet: https://docs.emmet.io/cheat-sheet/

Щоб працювати ще швидше — варто знати гарячі клавіші:

  • Ctrl+P — швидке відкриття файлу
  • Ctrl+` — відкрити термінал
  • Ctrl+/ — закоментувати рядок
  • Alt+Shift+F — автоматичне форматування коду

Повний список гарячих клавіш: keyboard-shortcuts-windows.pdf
